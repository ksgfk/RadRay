## srp framework (DONE, build-verified) — how the machine consumes game content
- srp::Scene (render/scene.h): AddRenderer(Renderer*)/RemoveRenderer (borrowed ptrs, vector<Renderer*>), AddLight()->Light*/RemoveLight(Light*) (owns unique_ptr<Light>), Renderers()/Lights() spans. srp::Light = plain data {Type,Direction,Color,Position,Intensity,Range,SpotInner/OuterAngle,CastShadow,ShadowDepthBias,ShadowNormalBias}. LightType{Directional,Point,Spot}.
- srp::Renderer (render/renderer.h ABSTRACT): MeshBatchElement BatchElement(); const VertexBufferLayout& GetVertexLayout(); const Matrix4f& WorldMatrix(); LayerMask()/CastsShadow()/IsVisible(); Material* GetMaterial(). MeshBatchElement{Vbv,Ibv,IndexCount,FirstIndex,VertexOffset}.
- srp::StaticMeshRenderer (render/static_mesh_renderer.h) final: ctor(StaticMesh*,primitiveIndex,firstIndex,indexCount,Material*); non-movable (owns semantic strings); SetWorldMatrix/SetMaterial; IsRenderable(). Builds geometry from RenderMesh->_drawDatas[primIdx] + MeshResource.Primitives layout.
- srp::Material (render/material.h ABSTRACT): GetShader()->srp::Shader*; MaterialKeywords()->KeywordSet; GetBlendMode()->BlendMode{Opaque,Masked,Transparent}; IsTwoSided(); AlphaCutoff(); Queue(); GetDescriptorSet(RootSignature*)->DescriptorSet*; MaterialSetIndex()->DescriptorSetIndex{1}.
- srp::StandardMaterial (render/standard_material.h) final: ctor(render::Device*, Desc). Desc{Shader* MaterialShader; BlendMode Blend; bool TwoSided; float Cutoff; string CBufferName="gMaterial"; vector<byte> CBufferData; vector<pair<string,render::TextureView*>> Textures; string SamplerName="gMaterialSampler"; DescriptorSetIndex SetIndex{1}}. GetDescriptorSet builds space1 reflection-free by FindParameterId(name), lazy+cached per rootSig. Textures must outlive material.
- srp::Shader (render/shader.h): ctor(ShaderId{uint64},name). AddPass(ShaderPassSource{string ShaderPath; string ShaderName; string VsEntry="VSMain"; optional<string> PsEntry="PSMain"; TagSet Tags(must contain "LightMode")}). HasPass/GetPassSource/ResolveTag/AddKeywordAxis. TagSet: vector<pair<string,string>> tags; add via .tags.push_back. (Need to confirm TagSet API — has .tags member + Find/LightMode helpers.)
- srp::ShaderVariantCache (render/shader_variant_cache.h): ctor(GpuSystem*). Get(Shader&,lightMode,KeywordSet)->const ShaderVariant*. Owns variant dedup; compile via GpuSystem.
- srp::RenderContext (render/render_context.h): ctor(GpuSystem*, ShaderVariantCache*). SetEncoder(GraphicsCommandEncoder*)/Encoder(). CreateRendererList(cull,draw,filter,rtFormats,pass)->RendererList. DrawRendererList(draw,list). Does filter→ResolveTag→variant→PSO→space1 set→per-object push→sort; record loop with redundant-bind skip.
- srp::RenderPass (render/render_pass.h ABSTRACT): Event()/ShaderTags()(WantedLightModes=vector<string>)/Filtering()(FilteringSettings{RenderQueueRange QueueRange; uint32_t LayerMask; Test(Renderer&)})/SortFlags()/PassKeywords()/RequiredPerObjectData()/RenderState(Material&)->MeshPassRenderState/RTFormats()->GpuRenderTargetFormats/ViewSet(SceneView&)->DescriptorSet*/ViewSetIndex()/PerObjectParamName()/PerObjectByteSize()/WritePerObject(span<byte>,Renderer&,SceneView&). - srp::DrawObjectsPass (render/draw_objects_pass.h) final: Desc{Event; WantedLightModes ShaderTags; FilteringSettings Filtering; SortingCriteria SortFlags; KeywordSet PassKeywords; PerObjectDataFlags RequiredPerObjectData; GpuRenderTargetFormats RTFormats; MeshPassRenderState RenderState; RenderStateProvider RenderStateFn; ViewSetProvider ViewSetFn; DescriptorSetIndex ViewSetIndex; string PerObjectParamName; uint32_t PerObjectByteSize; PerObjectWriter PerObjectFn}. - srp::RenderPipeline (render/render_pipeline.h): SetSetupPasses(fn(RenderPipelineExecutor&,SceneView&,CullingResults&)); RenderSingleCamera(ctx,cull); Render(ctx,span<cull>). Executor: EnqueuePass(RenderPass*), SortStable by Event, Execute.

## 6b CUTOVER DESIGN (decided)
OWNERSHIP: StaticMeshComponent OWNS its srp::StandardMaterial + srp::StaticMeshRenderer(s). On build (assets ready) it constructs StandardMaterial(device,Desc) from packed params+resolved texture SRVs+shader+blendmode, then one StaticMeshRenderer per mesh section referencing that material, registers each with srp::Scene. Material identity sharing lost (acceptable; SRP batches by variant/PSO not material).
- Component needs: srp::Shader* + render::Device* + BlendMode, set by spawner. Add SetRenderShader(srp::Shader*), SetRenderDevice(render::Device*), SetBlendMode(srp::BlendMode) to StaticMeshComponent. (Light/Camera need no device.)
- PrimitiveComponent rewritten: holds vector<unique_ptr<srp::Renderer>> (or single); OnRegister builds+AddRenderer; OnUnregister RemoveRenderer; OnTransformChanged calls SetWorldMatrix on each renderer. GetScene()->srp::Scene*.
- LightComponent rewritten: holds srp::Light* (from scene->AddLight()); OnRegister AddLight+push params; OnUnregister RemoveLight; setters mirror into *_light; OnTransformChanged sets Position/Direction from world transform.
- gltf ExportToScene: change signature to ExportToScene(World&, srp::Shader* shader, render::Device* device) (or add GltfExportContext). Passes shader+device+blendmode into each StaticMeshComponent. gltf load options DROP MaterialTemplate/DefaultMaterial old-material loading: primitives keep params+texture refs only; remove StreamingAssetRef<Material> Material field (or keep unused). Verify how blend mode currently derived → carry per-primitive blend in GltfPrimitiveDesc (add `srp::BlendMode Blend` or bool Transparent).
- example_imgui: builds sphere srp::Shader (single UniversalForward pass), StandardMaterial, sets on StaticMeshComponent; custom srp pass via DrawObjectsPass; RenderContext+pipeline. Replaces old SceneRenderer/RenderContext/RenderPipeline/RenderResourcePool usage.
- gltf_viewer: viewer-private LightBuffer (space0 builder over srp::Scene+srp::Light) + CSM ComputeDirectionalCascades + BuildAdditionalShadows (ported to srp). Passes: ShadowCaster/AdditionalShadow = custom srp::RenderPass subclasses (override Execute, multi-slice loop, BeginRenderPass per slice, CreateRendererList ShaderTags={"ShadowCaster"} + PassKeywords SHADOW_CASTER + per-object writes bias via gScene); PreZ/Base/Transparent = DrawObjectsPass configs (ViewSetFn returns space0 set; PerObjectFn writes SceneConstants MVP/Model/CameraPosition/Debug). Preview atlas = compute (keep as-is, own pipeline). RenderContext(gpu,&variantCache); SetEncoder per BeginRenderPass.
- PER-OBJECT for Base/Transparent/PreZ: SceneConstants = MVP(=ViewProj*World), Model(=World), CameraPosition(=EyePosition.xyz,0 / debug overlay sets Debug.x=7), Debug. PerObjectParamName="gScene", PerObjectByteSize=sizeof(SceneConstants)=2*64+16+16=160. ShadowCaster per-object: MVP=cascade.ViewProj, CameraPosition.xyz=dirToLight,w=depthBias, Debug.y=asfloat(normalBias).
